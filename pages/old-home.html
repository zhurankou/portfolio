My story

Currently, I'm a UX Designer on the Microsoft OneDrive team, working on cross-platform sharing and collaboration experiences.

In the past, I used to be an investigator, a turrets machine operator, a janitor, and a navy sailor, but I ended up doing what I've been always passionate about [design].

Besides the product design, I like to challenge myself with learning new technologies, such as React JS, React 360 and exploring prototyping tools like Framer X, Origami.

Also, I really like to work on design systems, productivity workflows, and collaboration between design and development.
